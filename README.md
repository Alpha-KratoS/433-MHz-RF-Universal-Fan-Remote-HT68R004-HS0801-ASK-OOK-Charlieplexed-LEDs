# -433-MHz-RF-Universal-Fan-Remote-HT68R004-HS0801-ASK-OOK-Charlieplexed-LEDs
This project implements a 433.92 MHz RF universal remote for ceiling fans using a Holtek HT68R004 MCU and HS0801 ASK/OOK transmitter. It supports 14 buttons, 15 LEDs, instant response, low power operation, reliable RF transmission, and scalable, industry-standard receiver logic.
